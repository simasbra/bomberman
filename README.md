# Bomberman

Bomberman game written in C# using Windows form. Single player and multiplayer mode available.

## Refactored to use design patterns:

1. singleton
2. factory
3. abstract factory
4. strategy
5. builder
6. prototype
7. observer
8. decorator
9. command
10. adapter
11. facade
12. bridge

13. template method
14. iterator
15. flyweight
16. composite
17. state
18. chain of responsibility
19. visitor
20. interpreter
21. mediator
22. memento
